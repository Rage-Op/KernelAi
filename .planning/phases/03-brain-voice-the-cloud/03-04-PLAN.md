<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| daemon -> Face (UDS NDJSON) | Inbound frames (speak/widget.data/ui.state) are decoded by the Face; a malformed line must not crash it |
| model output -> Face render | widget.data / reply text originate from model output (potentially injected) and are rendered |
| Face -> macOS TCC | The mic tap depends on the stable signed identity established in 03-03 |
| mic RMS -> cloud | High-frequency local signal — must stay Face-local (never the daemon) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-12 | Info disclosure | Markdown/remote-image exfil via the Face rendering model output | mitigate | The Face renders ONLY structured widget data — no auto-load of remote images/markdown/URLs from model output (RESEARCH Security Domain V5 / PITFALLS Pitfall 5). EventsWidget renders typed fields only. |
| T-03-13 | DoS | A malformed/spoofed inbound NDJSON frame crashing the Face | mitigate | KernelSocket mirrors server.ts: partial-frame-safe buffer + per-line Codable decode that tolerates a bad line (proven by FrameCodecTests malformed case). |
| T-03-14 | Spoofing/Tampering | IPC frame spoofing on the socket | mitigate | UDS is file-permission scoped (no network port); the Face connects to the user-owned socket path under ~/Library/Application Support/Kernel/. No remote attach surface. |
| T-03-15 | Info disclosure | Mic audio / RMS leaving the device | mitigate | RMS is computed and consumed entirely in the Face (CLOUD-03); only 16kHz PCM crosses to the LOCAL daemon for STT — nothing leaves the machine. |
| T-03-16 | Tampering/EoP | An out-of-bounds AVSpeech NSRange freezing/crashing the Stage | mitigate | StageController tolerates an invalid range (falls through to the time-based path); proven by StageControllerTests. |
