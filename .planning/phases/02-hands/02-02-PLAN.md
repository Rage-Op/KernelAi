<objective>
Land the Peekaboo GUI tool (HANDS-01, HANDS-02): a `tools/peekaboo.ts` MCP-client adapter that drives the brew-installed Peekaboo MCP server over stdio to capture the screen, click, type, navigate menus, list apps, and open + drive Mail. The only npm install here is `@modelcontextprotocol/sdk@1.29.0` (the MCP client), pinned EXACTLY (no caret), matching Phase-1 discipline.
